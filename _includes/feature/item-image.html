{% comment %} Item figure from image link. This include adds a figure to the page styled using bootstrap, https://getbootstrap.com/docs/4.5/content/figures/ It requires an "image" with the include, which is used to link to the image. E.G. --> {% include feature/item-image.html image="https://content.lib.washington.edu/tacomafoundingstoriesweb/image4.jpg" %} Options: - "image" = link to oustide image ; - "link" = link to somewhere with the image; - "caption" = by default the figure include automatically adds the title of the item from your metadata. The caption option allows you to manually add a different caption, or give the value false for none. - "width" = will use responsive sizing to set the % size on desktop (will be 100% on mobile), choose from "25", "50", "75", or "100" - "float" = will use responsive float utility to add float on desktop (will not float on mobile), choose from "left" or "right" {%- endcomment -%}
